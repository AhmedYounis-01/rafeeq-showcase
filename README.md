# Rafeeq Showcase

Rafeeq is a modern Islamic companion app that provides a complete Islamic experience with an elegant, responsive interface and support for both **Light** and **Dark** themes.

## ✨ Features

- 📖 Quran with Madinah Mushaf pages
- 🎧 Quran recitation
- 📚 Tafsir
- 🧭 Qibla Finder
- 📿 Digital Tasbeeh
- 🤲 Daily Adhkar & Du'a
- 🕌 Seerah Nabawya
- 🌙 Light & Dark Theme
- 📱 Responsive design for phones and tablets

## 🏗️ Built With

- Flutter
- Clean Architecture
- Bloc State Management
- Feature-based Architecture
